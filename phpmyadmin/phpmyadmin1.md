# membuat database
1. Server > Databases
2. Masukkan Database name dan pilih Collation.
3. Klik tombol Create

# melihat daftar table
1. Server > Databases
2. klik Structure

# mengubah Collation
1. Server > Databases
2. Klik nama database
3. Klik Operations di bagian atas
4. Scroll ke bawah
5. Pilih Collation dan ceklis **Change all tables collations**
6. Klik tombol Go
